Rules & Updates
  • Participation Rule: Each member should participate in only 2 rounds i.e. minimum and maximum 2 rounds
  • Venue Timing: Everyone should be present at the venue 15 minutes prior to the event start
  • Event Format: We have 5 offline rounds and 1 online round. Details of the online round will be released after the formation of teams
Day 1: Buzzer Round + AlgoArena (30 October)

Important: Each member can participate in maximum 2 rounds only. Plan your team wisely!

Buzzer Round

A fast-paced coding quiz with MultiBuzzer. Speed + accuracy define victory.

  • 40 coding/logic/aptitude questions
  • Team Size: 5 members
  • +3 correct, -2 wrong/delayed
  • No substitutions allowed
  • Venue: Seminar Hall | Duration: 1:30 Hours

AlgoArena

3–4 algorithm problems testing logic & problem-solving skills.

  • Team Size: 3 members
  • Duration: 60 minutes
  • Focus on efficient algorithms and data structures
Day 2: DSA Duel + Query Quest + CodeFusion (31 October)

Important: Each member can participate in maximum 2 rounds only. Plan your team wisely!

DSA Duel

Solve 3–4 data structure questions in 60 mins.

  • Team Size: 4 members
  • Scoring: 20 points per correct, max 60.
  • Focus on efficient data structures and algorithms

Query Quest

3–4 SQL/database problems testing database knowledge.

  • Team Size: 3 members
  • Scoring: 25 points per correct.
  • Database design and query optimization

CodeFusion

3–4 Python coding tasks.

  • Team Size: 3 members
  • Scoring: 25 points each, max 50.
  • Python programming and problem solving
